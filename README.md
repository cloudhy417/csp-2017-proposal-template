## Requirement
Latex should be installed.
Install Latex
```
$ sudo apt-get install texlive-full
```

## Start
```
$ mv ieee-template [your_name]
$ cd [your_name]
```
Then you can start to edit csp-paper.tex

## Build
```
$ make
$ xdg-open csp-paper.pdf
	to review your proposal in pdf format
```
